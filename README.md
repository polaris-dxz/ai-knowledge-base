# AI Knowledge Base

个人 AI Agent / AI Infra / AI Platform 知识库

> Building a long-term knowledge system for AI Agents, AI Infrastructure, and AI Platforms.

---

## Why

这个仓库用于沉淀和组织我在工作、学习和技术研究过程中积累的知识。

目标不是简单记录笔记，而是构建一个持续演进的技术知识体系，用于：

* 架构分析
* 技术调研
* 项目设计
* 技术写作
* 知识输出

最终形成自己的技术认知框架，并持续向社区输出高质量内容。

## Analysis Framework

对于系统或项目分析，统一采用以下框架：

1. What

   * 解决什么问题

2. Core Concepts

   * 核心对象与关系

3. Architecture

   * 系统架构

4. Data Flow

   * 请求链路与数据流

5. Design Decisions

   * 为什么这样设计

6. Trade-offs

   * 优势与局限

7. Reflection

   * 我的思考
